## Never build alone, join the community! 

### Get Started By: 

1. Join **over 40,000 other developers** in the Discord and GitHub Discussions on the [Community Links](#-community-links)

2. Attend the events, in Discord, hosted by our teams at Microsoft to give you the latest updates on Microsoft Foundry and **answer your questions.**

3. Complete our **free open source** [Courses](#-courses) to build fundamental skills in Gen AI and AI Agents.

4. Use one of our [Code Samples](#-code-samples) to start building **your own AI applications.**
